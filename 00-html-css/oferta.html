Desarrollador Full Stack

Acme Innovations | Remoto | Tiempo completo

Descipción del empleo

Buscamos un desarrollador Full Stack talentoso y motivado para unirse a nuestro equipo. El candidato ideal tendrá experiencia en el desarrollo de aplicaciones Web utilizando diversas tecnologías front-end y back-end. Será responsable de todo el ciclo de vida del desarrollo,desde la concepción y el diseño hasta las pruebas y el despliegue.

Responsabilidades

  • Desarrollar y mantener aplicaciones web utilizando React, Angular o Vue.js.
  • Diseñar e implementar APIs RESTful utilizando frameworks back-end como Node.js, Python (Django/Flask) o Java (Spring Boot).
  • Colaborar con otros desarrolladores, diseñadores y gestores de producto para definir y entregar nuevas funcionalidades.
  • Escribir código limpio, eficiente y bien documentado, siguiendo las mejores prácticas de la industria.

Requisitos

  • Licenciatura en informática o campo relacionado
  • Mínimo de 3 años de experiencia en desarrollo FullStack
  • Dominio de al menos un framework front-end (React, Angular, Vue.js) y un framework back-end (Node.js, Python, Java).
  • Experiencia con bases de datos relacionales y no relacionales (SQL, MongoDB).
  • Familiaridad con herramientas de control de versiones (Git) y metodologías ágiles.

Sobre la empresa

Acne Innovations es una empresa líder en tecnología que se centra en la creación de soluciones innovadoras para diversas industrias. Ofrecemos un entorno de trabajo dinámico y colaborativo, con oportunidades de crecimiento profesional y desarrollo personal. Valoramos la creatividad, la innovación y elcompromiso con la excelencia.
